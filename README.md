# elm-starter

## Ready to use setup
The workflow below is setup as npm script and bash script.

- [x] Build
    - Supported modes:
        - [x] debug build
        - [x] optimized build
    - Supported file types:
        - [x] Elm
        - [x] Scss
        - [ ] Typescript
- [x] Minify
    - [x] with file size report
- [x] Format
    - [x] use elm-format for Elm
    - [x] use prettier for scss
- [x] Watch file changes
    - [x] auto format
    - [x] auto build (debug mode)
